# CFMOTO Experience Kuban — Лендинг экскурсий на квадроциклах в Геленджике

Этот репозиторий содержит исходный код одностраничного лендинга для компании **CFMOTO Experience Kuban** — официального внедорожного полигона CFMOTO и сервиса проката/экскурсий на квадроциклах в Геленджике.

Сайт представляет услуги компании: экскурсии по горным маршрутам, прокат техники, подарочные сертификаты, корпоративы и дополнительные услуги.

## Структура проекта

├── index.html          # Основная HTML-страница (лендинг)
├── styles.css          # Основные стили сайта
├── scripts.js          # JavaScript-логика (Swiper, мобильное меню, аккордеоны и т.д.)
├── images/             # Все изображения (логотипы, фоны, фото маршрутов, команда и т.д.)

text

## Технологии

- **HTML5** — семантическая разметка, доступность (ARIA-атрибуты)
- **CSS3** — адаптивная вёрстка, Flexbox, Grid, кастомные свойства
- **JavaScript (vanilla)** — инициализация слайдеров, бургер-меню, аккордеон FAQ
- **Swiper.js** v12 — слайдеры для шапки, маршрутов, галереи фото
- **WebP** — современный формат изображений для оптимизации загрузки
- **Lazy loading** и **async decoding** для всех изображений
