Инструкция по использованию блока

Этот блок содержит все необходимые элементы для отображения видео с информацией:

  • Заголовок видео - основной заголовок в верхней части блока
  • Плеер - область для встраивания видео (YouTube, Vimeo, собственный видеоплеер)
  • Описание - подробная информация о содержании видео
  • Мета-информация - просмотры, лайки, комментарии, уровень сложности
  • Теги - ключевые слова для категоризации
  • Действия - кнопки для взаимодействия с видео
  • Информация об авторе - данные о создателе видео

Как встроить ваше видео: Замените блок .video-placeholder на код вашего видеоплеера (YouTube iframe, Vimeo iframe или собственный HTML5 video player).

Пример для YouTube: Вставьте iframe код вместо содержимого .video-player.
